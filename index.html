Children's book illustrations

2023 • illustrations• Clip Studio Paint

Created a series of children’s book covers using a soft, grainy illustration style inspired by artist Kate P. Bright, saturated colors were used to create visual engagement, while rounded character features emphasized warmth and approachability. This project reflects my interest in creating more engaging children’s books and served as practice in character design and expressive illustration for future graphic design and marketing work.
